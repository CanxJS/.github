## About CanxJS

**CanxJS** is a modern, high-performance backend framework built on the **Bun runtime**.  
It is designed for developers who want **maximum speed**, **clean architecture**, and **minimal configuration**.

CanxJS is ideal for building REST APIs, real-time applications, and scalable backend systems.
